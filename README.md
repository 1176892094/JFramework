1.协议库

程序集：JFramework.Kcp.dll

主要功能：传输协议，可靠Udp传输。

2.基础库

程序集：JFramework.dll

主要功能：引用池，事件池，序列化，压缩，加密。

3.引擎库

程序集：JFramework.Sdk.dll

主要功能：资源系统，资源热更，对象池，音效池，状态机，持久化，用户界面。

4.数据库 (控制台程序)

程序集：JFramework.Mysql.exe

主要功能：数据插入，数据查询，数据删除，数据更新。

5.服务器 (控制台程序)

程序集：JFramework.Lobby.exe

主要功能：大厅匹配，房间列表，主机踢人。

6.编辑器

程序集：JFramework.Editor.dll

主要功能：数据转化，数据导入，编辑器美化。

7.网络库

程序集：JFramework.Net.dll

主要功能：状态同步，帧同步，远程方法调用，网络同步变量。

8.调试器

程序集：JFramework.Log.dll

主要功能：引用数量，调试器窗口。
